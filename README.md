# 기계학습 중간고사 과제

# PART 1 — 분류(Classification) 모델 구현 및 분석

## 1. 과제 목표
- 실제 분류 데이터를 활용해 머신러닝 분류 모델을 설계하고 평가  
- 다양한 성능 지표 기반으로 모델을 비교·해석  
- scikit-learn 기반 전처리, 모델링, 튜닝 능력 평가  

---

## 2. 과제 수행 내용

### 2.1 데이터셋 선택
- 학번 규칙에 따라 **1개의 분류 데이터셋** 선택  
- 예시 데이터셋 (PDF 기준):  
  - Mushroom (독성 여부 이진 분류)  
  - Wine Variety (와인 품종 예측)  
  - Painting Genre (미술 작품 장르 분류)  

---

### 2.2 데이터 전처리
필수 전처리 항목:
- 결측치 처리  
- 범주형 변수 인코딩 (`get_dummies`, `OneHotEncoder`)  
- 스케일링 (`StandardScaler` 등)  
- 간단한 EDA 시각화  
  - 클래스 분포  
  - 주요 변수 분포  
  - 상관관계  

---

### 2.3 분류 모델 구현
**2개 이상**의 분류 알고리즘 사용:
- Logistic Regression  
- KNN  
- Decision Tree  
- Random Forest  

필수 조건:
- `train_test_split` 또는 `KFold`  
- `Pipeline` 또는 `GridSearchCV` 활용 가능  
- 코드에 충분한 주석 포함  

---

### 2.4 성능 평가
다음 모든 지표 사용:
- Accuracy  
- Precision  
- Recall  
- F1-score  
- Confusion Matrix  
- ROC Curve & AUC (이진 분류일 경우)

시각화 필수.

---

### 2.5 하이퍼파라미터 튜닝
- `GridSearchCV` 또는 `RandomizedSearchCV` 사용  
- 최적 파라미터 및 성능 개선 여부 분석  

---

### 2.6 제출물
- `.ipynb` 또는 `.py` 코드  
- 사용한 데이터  
- PDF 보고서  
- **모든 파일을 하나의 ZIP으로 압축 제출**  
- 파일 이름에 **한글 금지**, 윈도우에서 정상 열림 확인

# PART 2 — 회귀(Regression) 모델 구현 및 분석

- scikit-learn을 활용한 전처리, 모델링, 튜닝 능력 평가  

---
